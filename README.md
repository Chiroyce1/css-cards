# css-cards
Some good looking CSS cards I made

See them [here](https://chiroyce1.github.io/css-cards/).

## Features
- Comes in 4 colors
  - Blue
  - Purple
  - Yellow
  - Orange
- Auto theme switching 
- Closeable with an X button
- Responsive and works on mobile screens
